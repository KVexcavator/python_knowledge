```python
class Car:
    def __init__(self, brand, year):

        self.year = year

    def __repr__(self):
        return f"Car(brand={self.brand}, year={self.year})"

```
–ö–ª–∞—Å—Å —Å dataclass (–º–∏–Ω–∏–º—É–º –∫–æ–¥–∞, —Ç–æ—Ç –∂–µ —Ä–µ–∑—É–ª—å—Ç–∞—Ç!)
```python
from dataclasses import dataclass

@dataclass
class Car:
    brand: str
    year: int

car = Car("Tesla", 2022)
print(car)  # Car(brand='Tesla', year=2022)
```

–ï—Å–ª–∏ –Ω—É–∂–Ω–æ –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ —á—Ç–æ-—Ç–æ –∏–∑–º–µ–Ω–∏—Ç—å –ø–æ—Å–ª–µ —Å–æ–∑–¥–∞–Ω–∏—è –æ–±—ä–µ–∫—Ç–∞:

```python
class Product:

print(p)  # Product(name='Laptop', price=1234.57)
```
