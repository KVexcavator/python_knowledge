Моржовый оператор по-английски называется "walrus operator" walrus — это морж.
Он позволяет одновременно присвоить значение переменной и использовать его в выражении.

Пример:
```python
if (n := len([1, 2, 3])) > 2:
    print(f"Список длинный, длина = {n}")
```
Тут переменная n получает значение 3 
И сразу используется в условии > 2

Без моржового оператора:
```python
n = len([1, 2, 3])
if n > 2:
    print(f"Список длинный, длина = {n}")
```
