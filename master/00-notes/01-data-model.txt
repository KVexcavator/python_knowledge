API коллекций:
Есть три верхние абстакции ABC,у кахдой есть по одному втроенному методу:
- Iterable, __iter__, для поддержки for, распаковки и других видов итерирования;
- Sized, __len__, для поддержки встроенной функции len;
- Container, __contains__, для поддержки оператора in.
Абстрактный базовый класс Collection унифицирует(наследует) все три основных интерфейса, который должна реализовать любая коллекция.

Специализации Collection:
- Sequence, формализует интерфейс встроенных классов, в частности list и str;
- Mapping, реализован классами dict, collections.defaultdict и др.;
- Set, интерфейс встроенных типов set и frozenset.
